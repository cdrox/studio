# **App Name**: Metrics Maestro

## Core Features:

- Data Upload and Validation: Upload CSV/Excel files, validate data integrity, and provide data previews.
- Real-Time Metrics Dashboard: Display key CX metrics (AHT, CSAT, FCR, Escalations) with interactive charts, date range selectors, and drill-down capabilities. Calculations are done client-side.
- Anomaly Detection Engine: Detect statistical anomalies in CX metrics and visualize anomalous periods, with configurable sensitivity.
- Root Cause Analysis: Identify correlations between anomalies and patterns in the data to provide potential root causes for issues.
- AI Narrative Generation: Generate natural language narratives summarizing metrics, anomalies, and root cause insights. The narratives can be copied/exported.
- Industry-Specific Templates: Provide predefined metric configurations and industry-specific narratives.
- Data Export and Reporting: Export data and insights in text, JSON, and PDF formats for sharing and analysis.

## Style Guidelines:

- Primary color: Deep blue (#2E5AA5) to evoke trust and professionalism.
- Background color: Light gray (#F0F4F8) for a clean and modern look.
- Accent color: Teal (#39A7A7) to highlight important insights and calls to action.
- Body and headline font: 'Inter', a grotesque-style sans-serif font, is recommended for its modern, objective look and suitability for both headlines and body text.
- Use minimalist icons for metrics, filters, and export options.
- Dashboard layout with clear sections for data upload, metric display, anomaly visualization, and narrative generation.
- Subtle transitions and loading animations to enhance user experience.
